✓ Apple Music ✓ Qobuz ✓ Tidal ✓ Deezer ✓ Spotify ✓ SoundCloud ✓ Beatport ✓ Яндекс.Музыка
Скачивает музыку из подключённых сервисов вплоть до Hi-Res FLAC и Dolby Atmos. Токены доступа хранятся локально и за пределы устройства не передаются. Готовые файлы сохраняются в папку с музыкой и сразу доступны во встроенном плеере, поиске и очереди.

📁 Пути сохранения
Папки по качеству
alac/, flac/, 320/ … — любой сервис кладёт файлы в папку по формату, не по имени сервиса
После скачивания каждый файл перекодируется в выбранный формат (ffmpeg, метаданные+обложка переносятся). «Нативное» = отдаётся как есть. Из lossy в lossless качество не вырастет — это для единого формата.
Сохранять оригинал
Оставлять исходный файл рядом с результатом конвертации (по умолчанию оригинал удаляется). Игнорируется при «Нативное качество».
Запуск через go run (разработка)
Запускает исходный код командой go run main.go

⚙️ Движок Apple Music
zhaarey — Go, Docker wrapper + Apple ID  ·  gamdl — Python, cookies.txt  ·  AMD v2 — Python, ALAC/Atmos без Apple ID
Wrapper для ALAC/Atmos
Локальный — твой Docker-wrapper с премиум Apple ID (надёжно, без зависаний). Публичный — пул wm.wol.moe (без Apple ID, но бывают зависания). Авто — локальный для своего региона, публичный для чужих.
🟢 Docker не нужен для большинства сервисов — только для надёжного Apple ALAC / Atmos (режим «Локальный»). «Публичный» работает без Docker.
⚙️ «Локальный» требует Docker Desktop. Не установлен? Поставь с docker.com или переключись на «Публичный» (работает сразу).
Доступно только для движка zhaarey. AMD v2 — это всегда публичный wrapper, gamdl — cookies.

🔐 Widevine L3 CDM (опционально)
Lucida не дешифрует DRM-защищённые SC треки (миксы, приваты, большинство Go+ с 2024). Загрузи L3 device.wvd — Ripster будет сам делать Widevine handshake и сохранять AAC 256k.
статус: проверяю…
A. Свой L3 device.wvd (локальный)
Откуда взять: KeyDive — выгрузка из Android (рекомендуется), или wvdumper с Frida на эмуляторе. См. tools/widevine/README.md.
B. Удалённый wrapper (как wm.wol.moe для Apple)
Если у друга или у тебя на другой машине настроен device.wvd — введи URL его Ripster'а. Расшифровка пойдёт через него. Принцип: один host'ит → много пользуется.
C. Fallback на Deezer/Qobuz/Apple через ISRC поиск
Когда SC не качается — искать тот же трек на других сервисах автоматом
Полезно для релизов и популярных миксов (есть на Deezer/Apple). Бесполезно для приватных и эксклюзивных SC миксов — их нет нигде кроме SC.
